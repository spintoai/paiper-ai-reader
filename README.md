**논문, 이제는 쉽고 편하게 읽으세요!** AI가 논문을 완벽하게 번역해드립니다.

## 📥 다운로드

최신 버전은 항상 **[릴리즈 페이지](https://github.com/spintoai/paiper-ai-reader/releases)** 에서 받으실 수 있습니다.

### 🖥️ Windows 설치

1. **[릴리즈 페이지](https://github.com/spintoai/paiper-ai-reader/releases)** 로 이동
2. 최신 버전의 **Paiper-AI-Reader-Setup-...-win64.exe** 파일 다운로드
3. 다운받은 파일 실행 → 보안 경고가 뜨면 "추가 정보" → "실행" 클릭
4. 설치 완료! 이후 자동 업데이트됩니다

### 🍎 Mac 설치

1. **[릴리즈 페이지](https://github.com/spintoai/paiper-ai-reader/releases)** 로 이동
2. 최신 버전의 **Paiper-AI-Reader-...-arm64.dmg** 파일 다운로드 (Apple Silicon용)
3. dmg 파일 실행 → 앱을 Applications 폴더로 드래그
4. 설치 완료!

---

## 🚀 사용법

### 방법 1: PDF 파일로 읽기

1. 앱 실행 → **"PDF 파일 열기"** 클릭
2. 논문 PDF 선택
3. **"이 논문 번역하기!"** 클릭

### 방법 2: URL로 읽기

1. 논문 사이트 URL 복사
2. 앱 실행 → **"URL로 가져오기"** 클릭
3. URL 붙여넣기 → PMC 버전이 있으면 자동 제안
4. **"이 논문 번역하기!"** 클릭

### 방법 3: 직접 붙여넣기

1. 논문 본문을 복사
2. 앱 실행 → **"직접 입력"** 에 붙여넣기
3. **"이 논문 번역하기!"** 클릭

## ✨ 주요 기능

- **AI 번역**: 최신 고품질 AI 모델 기반, 문맥을 이해하는 자연스러운 번역
- **5가지 보기 모드**: 원문, 번역, 한영 대역, 문단 교차, 문장 교차
- **클릭 하이라이트**: 문장 클릭으로 원문/번역 동시 표시
- **온라인 리더**: 번역 논문을 PC, 스마트폰, 태블릿에서 열람 ([app.paiper-ai.com](https://app.paiper-ai.com))
- **그림·표·수식 지원**: PDF 그림/표 자동 추출, 수식 렌더링
- **PMC 자동 감지**: 저널 URL 입력 시 PMC 버전 자동 제안
- **PDF/HTML 내보내기**: 번역된 논문 저장
- **자동 업데이트**: 항상 최신 버전 유지
